# Customer Data Cleaning, Analysis and Reporting Automation Dashboard

## Project Overview

This project focuses on cleaning, preprocessing, analyzing, and reporting customer data using Python and Power BI. The dataset contains missing values, duplicate records, and inconsistent entries that are cleaned to improve data quality and generate meaningful insights.

<img width="1147" height="656" alt="Dashboard" src="https://github.com/user-attachments/assets/7822b7a5-d727-471d-a1a6-209bdd4c55f3" />


## Objectives

- Handle missing values
- Remove duplicate records
- Standardize categorical data
- Analyze customer demographics and purchase behavior
- Create automated reports and visualizations
- Build an interactive dashboard for business insights

## Tools and Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
- Power BI

## Dataset Features

- Customer_ID
- Age
- Gender
- City
- Purchase_Amount
- Segment
- Signup_Date

## Data Cleaning Steps

1. Loaded the raw dataset
2. Identified missing values
3. Handled missing data using appropriate techniques
4. Removed duplicate records
5. Standardized categorical values
6. Converted date columns into proper format
7. Validated and saved the cleaned dataset

## Dashboard Components

- Total Records
- Average Purchase Amount
- Customer Distribution by Gender
- Customer Distribution by City
- Customer Segmentation Analysis
- Purchase Trend Analysis
- Revenue Analysis
- Data Quality Metrics

## Key Insights

- Improved overall data quality by removing duplicates and handling missing values.
- Identified customer distribution across cities and segments.
- Analyzed purchase patterns and revenue contribution.
- Generated automated reports for better decision-making.
